## How It Works

The helper classes applies position helper property to an element.

- `u-position-{value}` - applies `position css property`

The value property is one of:

- `static`
- `relative`
- `absolute`
- `fixed`
- `inherit`

## Floats

Here is a list of float classes

- `u-float-left`
- `u-float-right`
- `u-float-none`
